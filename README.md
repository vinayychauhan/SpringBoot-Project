# 🛒 Spring Boot E-Commerce Website

A full-stack E-Commerce web application developed using **Java**, **Spring Boot**, **Spring Data JPA**, **JSP**, **HTML**, **CSS**, **Bootstrap**, and **MySQL**. The application provides a user-friendly shopping experience with secure authentication, product management, and database integration.

---

## 🚀 Technologies Used

### Backend
- Java
- Spring Boot
- Spring Data JPA
- Maven

### Frontend
- JSP
- HTML5
- CSS3
- Bootstrap

### Database
- MySQL

### Tools
- Eclipse IDE
- Git
- GitHub

---

## ✨ Features

- 👤 User Registration & Login
- 🔐 Secure Authentication
- 🛍️ Product Listing
- 🔍 Product Search
- 🛒 Shopping Cart
- 📦 Order Management
- 👨‍💼 Admin Panel
- 🗄️ MySQL Database Integration
- 📱 Responsive User Interface

## ⚙️ How to Run the Project

1. Clone the repository

```bash
git clone https://github.com/vinayychauhan/SpringBoot-Project.git
```

2. Open the project in Eclipse.

3. Create a MySQL database.

4. Update the database username and password in `application.properties`.

5. Run the Spring Boot application.

6. Open your browser and visit:

```
http://localhost:8080/
```
